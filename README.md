# 🚀 Amazon Finds - Viral Growth System

This project is a high-converting Amazon Influencer platform designed to maximize affiliate revenue through viral product discovery.

## 🛠️ Tech Stack
- **Frontend**: React 19, Vite, Tailwind CSS
- **Animations**: Motion (Framer Motion)
- **Icons**: Lucide React
- **Persistence**: LocalStorage (Wishlist & Analytics)

## 📂 Project Structure
- `src/App.tsx`: Main application logic, routing, and UI.
- `src/types.ts`: Data models, categories, and mock product database.
- `src/index.css`: Global styles and Tailwind configuration.

## 🚀 How to Run
1. Install dependencies: `npm install`
2. Start dev server: `npm run dev`
3. Open `http://localhost:3000`

## 🌐 Deploy Guide (10 Min)
1. **GitHub**: Push this code to a new GitHub repository.
2. **Vercel/Netlify**: Connect your repository to Vercel or Netlify.
3. **Environment Variables**: Add `GEMINI_API_KEY` if using AI features.
4. **Domain**: Connect your custom domain (e.g., `amazonfinds.com`) in the dashboard settings.

## 🤖 Updating Products
To update products without redeploying:
1. Modify `src/types.ts` with new product data.
2. For a truly dynamic system, replace `MOCK_PRODUCTS` with a `fetch()` call to a Google Sheets JSON endpoint or a CMS like Sanity.io.

## 📈 Traffic System
1. **Short-Form Video**: Post 2-3 TikToks/Reels/Shorts daily.
2. **Hook Formula**: "I found the one thing you didn't know you needed..."
3. **CTA**: "Link in bio" -> Points to this website.
4. **Consistency**: Use the "Trending Now" section to see what's working and double down.
